Overview

Advanced technologies have dramatically changed the ways that accounting professionals collect, analyze and interpret data. Embracing the prevalent implementation of advanced technologies, accounting is moving towards a STEM discipline and requesting accounting students be equipped with emerging technologies.

In response to the increasing requirements from the Association to Advance Collegiate Schools of Business (AACSB) and CPA exams on students' data analytics skills, this case provides you an opportunity to complete the data analytics process by analyzing Dillard's data using Structured Query Language (SQL) and Tableau.

The Case Study

About Dillard's Inc.

  • High-end department store chain based in Little Rock, Arkansas
  • Offers one-stop shopping for clothes, shoes, bags, accessories, home and beauty products
  • Founded by William T Dillard in 1938
  • Expanded through acquisition in the 1960s
  • Went public on the American Stock Exchange in 1969
  • Currently operates approximately 282 stores across 29 states
  • Texas holds the highest number of Dillard's stores with 57 establishments

Learning Objectives

SQL Data Extraction

Learn to write SQL queries to extract transaction data including amounts, prices, dates, and department information from Dillard's database.

Data Visualization

Create compelling visualizations using Tableau including geographic maps, word clouds, and box-and-whisker plots.

Analytics Process

Complete the full data analytics process from extraction to interpretation and communication of results.

Clustering Analysis

Perform clustering analysis to identify natural groups of cosmetic brands based on profit margins and sales data.

Key Deliverables

Required Outputs

  • SQL Queries: Extract transaction amounts, original prices, sale prices, transaction dates, costs, department descriptions, states, and transaction types
  • Geographic Analysis: Create maps showing average discount rates by state with 5% target rate
  • ETL Process: Extract, Transform, and Load data for analysis
  • Visualization Graphs: Develop comprehensive visual representations of the data
  • Interactive Dashboard: Build dashboard with word cloud filters
  • Analysis Report: Comprehensive report with key findings and insights
  • Presentation: Executive presentation for CEO and senior managers

Data Visualizations

Key Insights & Analysis

Discount Level Analysis

  • Christmas season discount patterns vary significantly by state
  • Iowa experienced significant discount rate jump from 9.63% to 19.03%
  • Target discount rate set at 5% for comparative analysis

Cosmetic Brand Performance

  • Natural clustering reveals distinct brand segments
  • Profit margins vary significantly across brands
  • Sales volume patterns indicate market preferences
