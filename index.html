ðŸŸ¢ Steps Followed by the Student

Step 1: SQL Data Analysis

The student first accesses the company's database and uses SQL queries to answer specific business questions:

Which brands have the highest sales?

SELECT BRAND, SUM(TRAN_AMT) as Total_Sales
FROM TRANSACTIONS t
JOIN PRODUCTS p ON t.PRODUCT_ID = p.PRODUCT_ID
WHERE DEPT_DESC = 'COSMETICS'
GROUP BY BRAND
ORDER BY Total_Sales DESC;

Which states offer the largest discounts?

SELECT STATE, 
       AVG((ORIG_PRICE - SALE_PRICE) / ORIG_PRICE * 100) as Avg_Discount_Rate
FROM TRANSACTIONS
WHERE ORIG_PRICE > 0
GROUP BY STATE
ORDER BY Avg_Discount_Rate DESC;

How does the return rate compare with industry average?

SELECT 
    COUNT(CASE WHEN TRAN_TYPE = 'RETURN' THEN 1 END) * 100.0 / COUNT(*) as Return_Rate,
    5.21 as Industry_Average
FROM TRANSACTIONS
WHERE DEPT_DESC = 'COSMETICS';

How did sales change throughout the holiday season?

SELECT 
    DATEPART(week, TRAN_DATE) as Week_Number,
    SUM(TRAN_AMT) as Weekly_Sales
FROM TRANSACTIONS
WHERE TRAN_DATE BETWEEN '2015-12-01' AND '2016-01-31'
    AND DEPT_DESC = 'COSMETICS'
GROUP BY DATEPART(week, TRAN_DATE)
ORDER BY Week_Number;
