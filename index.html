Abstract

Advanced technologies have dramatically changed the ways that accounting professionals collect, analyze and interpret data. Embracing the prevalent implementation of advanced technologies, accounting is moving towards a STEM discipline and requesting accounting students be equipped with emerging technologies.

In response to the increasing requirements from the Association to Advance Collegiate Schools of Business (AACSB) and CPA exams on students' data analytics skills, this case provides you an opportunity to complete the data analytics process by analyzing Dillard's data using Structured Query Language (SQL) and Tableau.

Specifically, this case provides an illustration of real-world data analysis, including the use of SQL to extract required data, the use of word clouds, geographic maps, and box-and-whisker plots to visualize data and outliers, an illustration of grouping data by clustering, and using interactive dashboards and narrative stories to communicate results.

Research Objective

We aim to use software tools that you are familiar with to inspire your interest in data analytics, helping you understand the enormous power of SQL and Tableau to query, aggregate, analyze and visualize massive data as you build your own data analytics mindset.

Your Role in This Case

Position: Manager of the Cosmetic Department

Task: The CEO asked you to prepare a report to summarize company's performance for the Christmas season.

Analysis Required:

  • Total sales analysis
  • Discount rate evaluation
  • Profit margin assessment
  • Sales growth rate tracking
  • Return rates comparison
  • Additional analysis proposal

Data Access: Microsoft SQL Server from Dillard's database

Goal: Impress the CEO with a professional report using your data analytics skills

Database Structure

Figure 1. UML for TRANSACT, SKU and DEPARTMENT tables.

Data Dictionary (Table 1)

TRAN_AMT Float Total transaction amount
ORIG_PRICE Float Original selling price
SALE_PRICE Float Selling price
TRAN_DATE Float Transaction dates
COST Float Cost of the products
DEPT_DESC String Brand names
STATE String State of stores
DEPTCENT_DESC String Department descriptions
TRAN_TYPE String Transaction types: purchase or return
TENDER_TYPE String Payment types

Five-Part Analysis Plan

1

Database Connection

Generate UML of the database from Microsoft SQL Server Management Studio and get sample data from each table.

2

Data Extraction

Write SQL queries to extract required fields and calculate DISCOUNT_RATE and PROFIT_MARGIN.

3

Data Visualization

Create geographic maps, word clouds, box-and-whisker plots, clustering analysis, and trend charts in Tableau.

4

Story Creation

Create a Tableau story highlighting important information on each graph for presentation.

5

Dashboard Development

Create an interactive dashboard enabling interaction between different graphs.

2. Deliverables

You need to prepare a presentation to communicate your analysis with the CEO and other senior managers. The static report is required to be submitted before the presentation.

1) Analysis Report

Format: 12-point Times New Roman font, single-spaced, one-inch margins

Content Requirements:

  • Analysis of discount level of different states
  • Trend of discount levels of states over weekdays
  • Popular cosmetic brands analysis
  • Natural clusters of cosmetic brands based on profit margin and total sales
  • Insightful discussions and suggestions under each graph

2) File Submissions

  • Static report (PDF format)
  • Tableau file (.twbx)
  • Presentation slides (5-10 pages)

3) Live Presentation

Present your analysis to the CEO (instructor) with real-time data access through Tableau. Be prepared to answer questions and edit your work to find additional answers.

Key Calculation Formulas

Discount Rate

DISCOUNT_RATE = (ORIG_PRICE - SALE_PRICE) / ORIG_PRICE

Profit Margin

PROFIT_MARGIN = (TRAN_AMT - COST) / TRAN_AMT

Return Rate

Industry Average = 5.21% (USPS, 2024)

3. Teaching Note

Data analytics has been widely integrated into accounting practices. With the increasing volume and velocity of big data, accounting professionals are expected to develop a diverse skill set to master emerging technologies.

Required Technologies:

Tableau Power BI Microsoft SQL Alteryx UIPath Python

Financial Accounting

SQL to efficiently extract, aggregate, and transform data from large datasets

Managerial Accounting

Data extraction and visualization tools to analyze KPIs and support data-driven decision-making

4. Conclusion

We believe this case presents an engaging and practical real-world scenario that enables students to access the data server of a familiar real-world department store company in the U.S. and analyze product brands they are familiar with.

Complete Data Analytics Process

Our comprehensive case shows students the whole picture of data analytics, starting from asking business questions, collecting data, analyzing data, visualizing data, and communicating results.

Data Analytics Mindset

The case helps students practice data analytics skills and build a data analytics mindset, understanding how companies use SQL and Tableau to generate valuable information and make data-driven decisions.

Flexible Implementation

Instructors can use this case directly in accounting data analytics or accounting information systems classes, or adapt it for financial accounting or managerial accounting courses.
